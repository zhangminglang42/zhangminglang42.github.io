## Research Projects

### AI-Enhanced Weather Forecasting System
*2023-Present*

A comprehensive deep learning framework for improving numerical weather prediction accuracy through post-processing and data assimilation.

**Key Features:**
- Ensemble forecast calibration using neural networks
- Multi-modal data integration (satellite, radar, station observations)
- Real-time inference pipeline for operational forecasting
- Uncertainty quantification with Bayesian deep learning

**Technologies:** PyTorch, TensorFlow, xarray, Dask, Kubernetes  
**Status:** Active development, deployed in experimental operations at Guangdong Meteorological Bureau  
[GitHub Repository](https://github.com/zhangminglang42/ai-weather-forecasting)

### Climate Model Emulator
*2022-Present*

Neural network-based emulator for accelerating climate model simulations while maintaining physical consistency.

**Key Features:**
- 1000x speedup compared to traditional climate models
- Physics-informed neural network architecture
- Multi-scale modeling capability (global to regional)
- Uncertainty propagation through ensemble methods

**Technologies:** JAX, Neural Operators, PyTorch, NetCDF  
**Status:** Research prototype, validated against CESM and WRF simulations  
[Project Page](https://zhangminglang42.github.io/climate-emulator/)

### Extreme Weather Detection Platform
*2023-Present*

Deep learning system for automatic detection and tracking of extreme weather events from satellite imagery.

**Key Features:**
- Real-time detection of tropical cyclones, thunderstorms, and heatwaves
- Multi-sensor fusion (visible, infrared, microwave)
- Early warning system with probabilistic forecasts
- Interactive visualization dashboard

**Technologies:** PyTorch Lightning, FastAPI, React, Docker  
**Status:** Operational prototype, integrated with CMA satellite data feed  
[Demo](https://demo.extreme-weather.ai)

## Datasets

### China Meteorological Reanalysis Dataset (CMRD)
*A high-resolution reanalysis dataset for China region*

**Description:**
- 10km spatial resolution, hourly temporal resolution
- 40-year period (1980-2020)
- Multiple variables (temperature, precipitation, wind, humidity)
- Quality-controlled with station observations

**Access:** Available through National Meteorological Information Center  
[Documentation](https://cmrd-docs.readthedocs.io/)

### AI-Ready Weather Dataset
*Curated dataset for machine learning applications in meteorology*

**Description:**
- Preprocessed and normalized weather data
- Multiple forecasting tasks (nowcasting, medium-range, seasonal)
- Benchmark tasks with evaluation metrics
- Tutorial notebooks and baseline models

**Access:** Open access via Hugging Face Datasets  
[Hugging Face](https://huggingface.co/datasets/zhangminglang42/ai-weather)
